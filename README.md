# ryuya-inmu-chain

CPU対戦でルールを覚えるための最小プロトタイプです。

## 内容

- 5 x 3 の盤面
- 手札からキャラを配置
- 矢印方向への自動攻撃
- 味方同士のチェイン攻撃
- リーダーHPと勝敗判定
- 簡易CPU
- 必殺技ゲージ
- アーケード風の盤面UI
- チェイン・ダメージ・コンボ演出
- INMU PORTAL寄りの黒基調ネオンUI
- 発光リンク線と爆発エフェクト
- 手札・配置前の矢印プレビュー
- 3Dポータルマスコットのアイドルアニメーション

## 起動

`index.html` をブラウザで開いてください。
