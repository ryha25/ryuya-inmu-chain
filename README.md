# ryuya-inmu-chain

INMU PORTAL風のCPU対戦プロトタイプです。5 x 3の盤面にキャラクターを配置し、矢印方向のリンクとチェイン攻撃で相手リーダーを倒します。

## 内容

- 5 x 3の盤面
- 手札からキャラクターを配置
- 矢印方向への自動攻撃
- 味方同士のチェイン攻撃
- リーダーHPと勝敗判定
- CPU対戦
- 必殺技ゲージ
- INMU PORTAL寄りの黒基調ネオンUI
- 発光リンク線、着弾、ダメージ、コンボ演出
- 手札と配置前の矢印プレビュー
- 3Dポータルマスコットのアイドルアニメーション
- 盤面駒のキャラ別3D表示
- 直線上の敵をまとめて攻撃する貫通タイプ
- FIGHTカットイン、上部スキルバナー、マス番号ガイド
- 3Dキャラ用の青/赤/金ペデスタルと着地アニメーション
- 奥へ続く3Dアリーナ背景
- 配置、攻撃、必殺技に合わせたズーム/チルトのカメラワーク
- 参照画像に寄せたリーグ風HUD
- 上部HPバー、中央TURN/TIME表示、左右情報パネル、下部カードドック
- サイバーシティ背景画像

## 起動

`index.html`をブラウザで開いてください。
